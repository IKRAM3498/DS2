EL-WALI IKRAM / 24010354 / 
Base de données : Maladie cardiaque (Heart Disease)

Date de don : 30 juin 1988
Origine : 4 sources — Cleveland, Hongrie, Suisse et VA Long Beach

Caractéristiques générales

* Type de données : multivariées
* Domaine : santé et médecine
* Tâche principale : classification
* Types de variables : catégorielles, entières et réelles
* Nombre d’observations : 303
* Nombre de variables : 13

Description
Cette base de données contient à l’origine 76 attributs, mais les recherches publiées utilisent généralement un sous-ensemble de 14 attributs.
Parmi les quatre bases, celle de Cleveland est la plus utilisée par les chercheurs en apprentissage automatique.

L’objectif du jeu de données est de déterminer la présence ou l’absence d’une maladie cardiaque chez un patient.
La variable cible (goal) prend des valeurs entières de 0 à 4 :

* 0 → absence de maladie cardiaque
* 1 à 4 → présence de la maladie (de divers degrés)

Les travaux de recherche se concentrent généralement sur la classification binaire : distinguer la présence (1–4) de l’absence (0) de la maladie.

Les noms et numéros de sécurité sociale des patients ont été supprimés et remplacés par des valeurs factices.

Un fichier a été pré-traité (celui de Cleveland), tandis que les trois autres restent non traités.
Des informations supplémentaires sur les coûts de test (fournies par Peter Turney) se trouvent dans le dossier "Costs".
